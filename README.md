# 二维码生成器

## 前言

2024年的3月份，我还是一个刚刚适应大学生活的菜鸟，想做些什么又不知道从何开始奋斗，这一段懵懂的时期持续了很久。这个项目就是懵懂时期的产物，如今自己逐渐明白自己应该做什么，弄清楚了自己的方向，却尤想起自己参加雷军班三次选拔答辩的时候，一个评委老师问及“你会回顾你的项目并做出改进吗？”为了应付问题，我当然回答“是”，尽管实际上并没有，原因有二：自己在大学一直在进步，不由觉得以前的作品与现在自己能做的相差巨大，有一种割裂感，比如那时的项目ide还是并不习惯的vs2022；学业生活愈来愈忙，很难抽身做自己的事儿了，特别是这种自己觉得意义不大的事儿。今天重返修缮这个项目，怎么想都有些突然和矛盾，或许我该把更多时间放在该做的事儿上——为龙芯杯做准备。或许是因为看到下一届同学正为大作业的事儿熬夜苦赶，重燃了内心写项目的热情，或许是因为复习周难得闲可以做自己想做的事儿。但是我更喜欢这个诗意的答案：只是因为觉得自己应该去做，而去做。

## 项目介绍

### 功能介绍

二维码生成器。支持所有包含ASCII字符、汉字和日文信息的加密，并输出为一张二维码图片。

### 项目架构

```
QRcode(remade)/
├── include/
│   ├── Parser.h
│   ├── Drawer.h
│   └── Utils.h
├── src/
│   ├── Parser.cpp                  # 输出二维码信息的比特流
│   ├── Drawer.cpp                  # 根据比特流输出 bmp 后缀图像
│   └── QRcodeGenerator.cpp         # 调用接口示例
├── utils/
│   ├── encodeConversion.cpp        # Utf-8 到 sjis 和 GBK 编码的转换
│   └── reedSolomon.cpp             # 里德所罗门编码的实现
├── out/
│   ├── example.bmp                 # 本项目生成的二维码, 指向本项目
│   └── 1.bmp                       # 测试生成的二维码
├── resources/
│   ├── bitMode.txt                 # ASCII 码字符对照表, 用于参考, 未使用
│   └── sjis.tbl                    # sjis 编码对照表, 用于参考, 未使用
└── .gitignore
```

### 调试

```cmd
g++ -O2 src\QRcodeGenerator.cpp src\Parser.cpp src\Drawer.cpp utils\encodeConversion.cpp utils\reedSolomon.cpp -o main.exe
```
